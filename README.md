# متجر غذاء (Gidha Store)

متجر إلكتروني للمنتجات الغذائية الطبيعية والعضوية. تم بناء واجهة المستخدم (UI) باستخدام تقنيات الويب الأساسية وتصميم متجاوب بالكامل.

## الميزات التقنية
- HTML5 & CSS3 (Vanilla)
- Vanilla JavaScript بدون مكتبات خارجية
- Mobile First Responsive Design (متوافق مع الشاشات الصغيرة والكبيرة)
- دعم كامل للغة العربية (RTL) باستخدام الخصائص المنطقية (Logical Properties)
- لا يتطلب Build Tools (No Webpack/Vite) لسهولة النشر المباشر.

## طريقة المعاينة محلياً
يمكنك فتح المشروع بسهولة عن طريق:
1. استنساخ المستودع (Clone) أو تحميل الملفات.
2. فتح ملف `index.html` الموجود في المجلد الرئيسي باستخدام أي متصفح (يفضل Chrome أو Firefox).
3. أو استخدام إضافة مثل **Live Server** في VS Code لفتح الموقع وتصفحه بشكل سلس.
